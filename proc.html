Processos:

 

O grupo usa como processo de desing a metodologia ágil scrum. Usa-se a ata da cadeira de Projetos, esssa que foi baseada na metodologia, para alinhar e organizar as atividades entre o grupo. Paralelo a isso se usa o Trello para desenvolvimento das pesquisas. Usa-se também o Google Drive como hub de compartilhamento da equipe.

Em Ciência da Computação usamos o GitHub para desenvolvimento dos softwares requiridos. Também usamos as metodoligias contempladas na cadeira de Fundamentos de Desenvolvimento de Software.

Metodologia:

 

O grupo trabalha com processo de desenvolvimento incremental, sempre alinhando com o cliente para uma maior satisfação do cliente com o resultado final.
